Exit Smartdesigner

Download this script here

This script adds an extra button to the top of the page when editing in Smartdesigner. Clicking the button exits Smartdesigner, and returns you to the page you were editing without being in editing mode. This is similar to the behavior when editing a Canvas page in the regular editor. Once you exit editing mode, you are taken to the page you were currently editing.
