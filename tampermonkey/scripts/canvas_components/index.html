Canvas components

Download this script here

This script adds an extra button to the toolbar in the Rich Text Editor in Canvas when editing pages. The icon is a jigsaw puzzle piece (se red box in image below).

When clicking the icon, a new dialog box will open up showing the various components you can insert. Find the one you want and click the corresponding button to insert it in the editor at the position of the cursor. The image below shows what the dialog box can look like.
